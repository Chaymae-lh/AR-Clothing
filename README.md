#  AR Clothing – Boutique de vêtements en ligne

##  Description du projet
**AR Clothing** est un site e-commerce simple et moderne destiné à une boutique de vêtements .  
Le site permet de présenter des produits, de filtrer par catégories, de rechercher des articles et de commander facilement via **WhatsApp**.  
Un **espace administrateur sécurisé** permet au vendeur d’ajouter, modifier et supprimer des produits depuis un téléphone ou un ordinateur.

Le projet est conçu pour être :
- 100% gratuit (hébergement + outils)
- Responsive (PC, tablette, mobile)
- Facile à utiliser pour le vendeur

##  Fonctionnalités principales

###  Côté client
- Page d’accueil avec section **Hero**
- Affichage des produits depuis la base de données
- Filtrage par catégories
- Recherche dynamique
- Page détail produit
- Bouton **Commander via WhatsApp**
- Page contact avec réseaux sociaux
- Design responsive (menu hamburger sur mobile)

###  Côté administrateur
- Authentification admin (login / logout)
- Tableau de bord (dashboard)
- Ajout de produits (nom, prix, ancien prix, image, description, catégorie)
- Suppression de produits
- Upload d’images
- Sécurité par sessions PHP
